## Riesgos Conocidos

1. **Microsoft Edge**: Requiere Microsoft Edge instalado para exportar PDF (pre-instalado en Win10/11 modernos). Si no está, el PDF no se puede generar pero el HTML queda disponible.

2. **Permisos de administrador**: Algunas funciones requieren permisos de admin (punto de restauración, Windows\Temp). La app maneja esto con advertencias claras.

3. **Frecuencia de puntos de restauración**: Windows limita a 1 punto por día. La app maneja este error y muestra mensaje informativo.

4. **WMI en equipos remotos**: Algunos datos de hardware pueden no estar disponibles en todos los equipos.

5. **Logo como ruta absoluta**: El logo se guarda como ruta absoluta. Si se mueve la imagen, se pierde la referencia. (Documentado como TODO para futura mejora)

---

## Limitaciones Actuales

1. **Sin DI (Dependency Injection)**: Los servicios se instancian directamente en MainViewModel. Pendiente implementar DI real.

2. **Sin logging estructurado**: Serilog está configurado pero no se usa activamente en todos los servicios.

3. **Sin tests de integración**: Solo tests unitarios de modelos. Los tests de servicios requieren entorno Windows real.

4. **Sin instalador**: La app es portable pero no tiene instalador profesional.

5. **Sin actualizaciones**: No hay mecanismo de actualización automática.

6. **Sin multi-idioma**: Solo español.

---

## Recomendaciones antes de v0.2

1. **Implementar DI**: Configurar Microsoft.Extensions.DependencyInjection en App.xaml.cs
2. **Configurar Serilog**: Activar logging estructurado en todos los servicios
3. **Tests de integración**: Agregar tests que ejecuten servicios reales en entorno controlado
4. **Performance profiling**: Medir tiempos de escaneo y generación de informes
5. **Validación de UI**: Probar en resoluciones bajas (1280x720)
6. **Backup de configuración**: Agregar exportación/importación de configuración
